# Autonoma ERP — AI Context Instructions

> READ `BOS_DEVELOPMENT_GUIDE.md` IN THIS DIRECTORY BEFORE WRITING ANY CODE.

## Quick Rules

1. Use `BOSFormDialog`, `BOSFormSection`, `BOSTextField`, `BOSDataTable` from `ui-component/bos`
2. Use `useBOSValidation` hook — never `alert()` or `window.confirm()`
3. Use `ConfirmDeleteDialog` for all delete operations
4. Use `useKeyboardShortcuts` with `shortcutTooltip()` on every page
5. Use `openSnackbar` from `store/slices/snackbar` for all notifications
6. Use `setFilterConfig` from `store/slices/search` on every list page
7. All style tokens come from `ui-component/bos/BOSStyles.js` — no inline darkStyles
8. Every list page wraps in `MainCard` with icon+title and action buttons
9. Button colors: Save=Green, Edit/New=Blue, Delete=Red, Cancel=Gray
10. Double-click on table row = open edit dialog

## File Locations

- BOS Components: `src/ui-component/bos/`
